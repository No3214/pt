# Frontend Design Skill

This skill guides creation of distinctive, production-grade frontend interfaces that avoid generic AI slop aesthetics.

## Design Thinking

Before coding, commit to a BOLD aesthetic direction:
- Purpose: What problem does this interface solve?
- Tone: Pick an extreme - brutally minimal, maximalist, retro-futuristic, luxury/refined, editorial/magazine, brutalist/raw, art deco, soft/pastel
- Differentiation: What makes this UNFORGETTABLE?

## Frontend Aesthetics Guidelines

- Typography: Distinctive fonts. NEVER Arial, Inter, Roboto. Pair display + body fonts.
- Color: CSS variables. Dominant colors with sharp accents. No timid palettes.
- Motion: Framer Motion for React. Staggered reveals, scroll-triggered, spring physics.
- Spatial Composition: Asymmetry, overlap, diagonal flow, grid-breaking.
- Backgrounds: Gradient meshes, noise textures, geometric patterns, grain overlays.

## PT Project Fonts and Colors

- Display: Cormorant Garamond (serif)
- Body: Outfit (sans-serif)
- Primary: #C2684A (terracotta)
- Secondary: #7A9E82 (sage)
- Accent: #5e8fa8 (ocean)
- Sand: #D4C4AB

## Anti-Patterns

- Generic AI aesthetics (Inter, Roboto, purple gradients on white)
- Predictable layouts, cookie-cutter design
- Same font across all generations
