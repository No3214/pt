# SEO Optimizer Skill

Search engine optimization for PT platform.

## On-Page SEO Checklist
- Title tag: under 60 chars, primary keyword near start
- Meta description: 150-160 chars, compelling, keyword included
- H1 with primary keyword, proper H2/H3 hierarchy
- URL slug: readable, keyword-rich (/blog/react-hooks-guide)
- Images: WebP, descriptive alt text, explicit dimensions, lazy loading
- 3-5 internal links per 1000 words
- Canonical tag set correctly

## Technical SEO
- Schema markup: Article, FAQPage, LocalBusiness, BreadcrumbList
- robots.txt: Disallow /admin/, /api/; Allow public routes
- XML sitemap with lastmod dates
- Canonical tags to prevent duplicate content
- Open Graph + Twitter Card meta tags

## Core Web Vitals Targets
- LCP: under 2.5s (optimize images, use CDN)
- FID: under 100ms (defer non-critical JS)
- CLS: under 0.1 (set image dimensions, reserve space)

## PT-Specific SEO
- Turkish + English meta tags per language
- hreflang tags for multilingual pages
- Schema: LocalBusiness for Ela PT
- FAQ schema on /faqs section
- Cloudflare CDN for fast global TTFB

## Content Strategy
- Long-tail keywords: voleybol antrenman programi, sicrama egzersizleri
- Blog posts: 1500-2500 words, comprehensive
- Topic clusters: training, nutrition, volleyball, wellness
