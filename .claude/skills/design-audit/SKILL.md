# Design Audit — Ela Ebeoğlu PT
## Brand: Terracotta #C2684A, Sage #7A9E82, Coast #4A6D88, Bg #FAF6F1
## Principles (from Emil Kowalski + Impeccable + Taste skills)
1. Spring physics animations, not linear easing (animations.dev)
2. Hover/focus states on EVERY interactive element
3. No design should look "default" or "template-like"
4. 44x44px min touch targets. 4px spacing scale.
5. WCAG 2.1 AA: 4.5:1 contrast, focus indicators, aria labels
6. Warmth & Approachability: soft shadows, generous spacing
## Audit: hierarchy, color, typography, spacing, alignment, responsive, micro-interactions, loading/empty/error states
## Elevation: shadow-sm(rest) shadow-md(hover) shadow-2xl(modal)
## Animation: ease [0.22,1,0.36,1], viewport once, staggerChildren
