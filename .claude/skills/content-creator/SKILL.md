## Content Types

### Blog SEO 2026
- 1500–2500 word (long-form ranking)
- Structured data: Article schema + FAQ + HowTo
- `<h1>` primary keyword start
- Internal link 3-5/1000 word
- WebP/AVIF hero image + CTA
- E-E-A-T: author bio (Ela, 12y exp + sertifika)
- Keyword cluster: voleybol antrenman, sıçrama programı, VBT, periodization

### Instagram / Reels / TikTok
- **Reels (15-60s)** — exercise form demo + caption hook
- **Carousel** — before/after metric + protocol breakdown
- **Story** — daily win, client PR, Q&A
- **Hashtag stack** #voleybol #antrenman #fitness #strengthcoach #performans #beslenme
- **CTA in bio** — WhatsApp link + landing

### Email Sequence (2026 pattern)
- **Welcome (Day 0)** — intro Ela + free PDF guide
- **Educate (Day 2,5,9)** — training tip + nutrition myth bust
- **Case study (Day 14)** — client transformation + metric
- **Conversion (Day 21)** — limited slot + testimonial
- Personalization token (first_name, city, sport)

### Landing Copy
- **Hero H1**: bold claim + social proof stat (3-line max)
- **Benefit**: results-focused (+12cm jump, -8% body fat)
- **Testimonial**: named athlete + specific metric + photo
- **CTA**: WhatsApp direct, free consultation
- **Proof strip**: brand/club logo

### Short-Form Video Script
- **Hook (0-3s)** — problem statement ("Voleyboldan ağrı mı geliyor?")
- **Insight (3-15s)** — science + analogy

## Writing Rules
- `t.xxx` i18n (13 dil uygulama)
- **Metric > adjective** ("+12cm" not "önemli artış")
- **Active voice**, kısa cümle
- **Turkish**: sade, anlaşılır, motive edici — doktor dili YASAK
- **Avoid filler**: "olarak", "aslında", "bu bağlamda"
- **CTA clarity**: tek aksiyon, belirsizlik YASAK

## SEO 2026
- Title ≤60 char, primary kw start
- Meta 150-160 char compelling
- URL readable (`/blog/voleybol-sicrama-antrenman`)
- Alt text descriptive + kw
- Schema: Article + FAQPage + LocalBusiness + BreadcrumbList
- hreflang 13 dil
- Canonical tag duplicate önle
- Core Web Vitals v4: LCP ≤2.0s, INP ≤150ms
